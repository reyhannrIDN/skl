## 🎬 Animation System

### Background Animations
```
Floating Orbs:
✨ Blue orb (top-left): 0s delay
✨ Amber orb (top-right): 2s delay
✨ Purple orb (bottom-left): 4s delay
⏱️ Duration: 7 seconds each
🔄 Infinite loop

Motion:
• Translate: Moving position
• Scale: Grow/shrink (0.9-1.1)
• Opacity: Fade in/out (0.2-0.4)
```

### Entry Animations
```
Elements:
📌 Badge: fade-in (1000ms)
🎯 Heading: slide-in-from-bottom-8 (1000ms delay 100ms)
📝 Subheading: slide-in-from-bottom-8 (1000ms delay 200ms)
🔍 Search: fade-in zoom-in-95 (1000ms delay 300ms)
📋 Result: slide-in-from-bottom-12 (700ms)
```

### Interaction Animations
```
Hover Effects:
🎨 Card: Border brightens, glow appears (300-500ms)
🔘 Button: Shadow expands, text glows (300ms)
🎯 Icon: Scale 110%, rotate -6deg (300ms)
🔗 Link: Underline grows, arrow translates (300ms)
📌 Badge: Color transitions, border changes (300ms)
```
