## 🔧 Customization Guide

### Change Primary Color (Blue)
Replace `blue-500`, `blue-400`, `blue-300` with your color:
```
bg-indigo-500, text-indigo-400, hover:text-indigo-300
```

### Change Dark Tone
Replace `slate-950`, `slate-900`, `slate-800` with:
```
gray-950, gray-900, gray-800 (cooler)
zinc-950, zinc-900, zinc-800 (neutral)
```

### Adjust Animation Speed
Change `duration-1000` to `duration-500` (faster)
or `duration-1500` (slower)

### Modify Glow Effect
Change `shadow-blue-500/40` opacity:
```
shadow-blue-500/20 (subtle)
shadow-blue-500/60 (intense)
```
